# Chat-Based Layout Agent

A chat-based AI layout editing tool that updates design JSON using natural language instructions.

## Features

- Chat interface
- Layout JSON updates
- Wireframe preview
- Aspect ratio conversion
- Element movement
- Text resizing

## Tech Stack

Frontend:
- React
- Vite

Backend:
- Node.js
- Express

## Run Project

### Backend

```bash
cd server
npm install
node index.js
```

### Frontend

```bash
cd client
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```
